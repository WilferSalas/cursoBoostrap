Clases

container

El el contenedor principal y nos permite centrar el contenido.

row

Dentro de cada container tenemos lo que son las filas que son grupos de columnas horizontales.

col

Son las columnas dentro de las cuales se agrupa el contenido

Columna

Para especificar el numero de columnas que se desea tener se hace asi:

class="col-n" Donde n es el numero de columnas

Si no se hace esto, por defecto Bootstrap calcula el numero de columnas disponible y usa ese tamaño

Nota:

Dentro de cada "container" debe haber una clase "row" que son las filas y a su vez, una clase "col" que son las columnas. Jerarquia: container > row > col

Tamaños de las columnas:

.colTamaño extra pequeño - Smartphone vertical (Menos de 544px)

.col-sm Tamaño pequeño - Smartphone vertical (Mas de 544px y menos de 768px)

.col-md Tamaño mediano - Tablest (Mas de 768px y menos de 992px)

.col-lg Tamaño largo - Computadoras (Mas de 992px y menos de 1200px)

.col-xl Tamaño extra largo - Computadoras (Mas de 1200px)

Columnas automaticas

Esto nos permite indicarle a la columna que use solo el espacio que necesite

col-auto

Solo usa lo que necesita

Tambien se le puede indicar que sea automatico luego de cierto tamaño.

col-md-auto

Usa todo si se reduce el tamaño de la pagina

Separador

Bootstrap permite utilizar una clase para crear un separador entre columnas

class="w-100" donde 100 es el tamaño en % de pantalla que ocupara el separador

Alineando columnas verticalmente

En la version 4 es posible alinear las columnas de forma vertical utilizando flexbox y sin la necesidad de usar CSS. Esto se hace de la siguente manera

Para alinear la columna al inicio de la fila: class="row align-items-start"
